REAL-WORLD ROBOTICS · 2025—2026

Real-world Imitation Learning

ACT & Diffusion Policy on Franka Panda

ALOHA 중심으로 활용되던 ACT를 Franka Panda에 적용할 수 있도록 teleoperation, 데이터 수집, 학습, 추론, 제어 환경을 확장했습니다. Temporal Ensemble 대신 다항식 보간 기반 제어를 적용해 추론 부담과 궤적 불연속을 줄였습니다.

같은 실험 워크스페이스에 U-Net·DiT 기반 Diffusion Policy를 구현하고 문 열기, Pick-and-place 등 long-horizon task에서 실로봇 성능을 비교했습니다. MuJoCo 기반 Sim2Real 워크스페이스도 함께 구축했습니다.

REINFORCEMENT LEARNING · 2024—2025

Mobile Manipulation RL

Single PPO → Hierarchical Reinforcement Learning

모바일 매니퓰레이터의 문 열기 작업을 위해 보상 함수, 충돌 조건, gripper 제어와 종료 조건을 수정하며 단일 PPO 정책을 개선했습니다. 기구학·동역학과 접촉 조건을 고려해 실험 환경을 단계별로 디버깅했습니다.

여러 단계가 연결되는 long-horizon task에서 단일 정책이 갖는 한계를 확인한 뒤, 상위 task generator와 하위 controller로 구성된 계층형 정책으로 확장했습니다. iGibson 결과를 재현하고 MuJoCo로 환경을 변환해 복제 가능성을 검증했습니다.

SIMULATION · 2026

Scalable Neural Simulator

Articulated rigid-body simulation and Sim2Real evaluation

관절형 강체를 위한 Neural Simulator 개발에서 technical ablation을 담당하고, 충돌·수치 적분·상태 전이 등 물리 시뮬레이션 요소를 분석했습니다.

Isaac, RaiSim, MJX, MuJoCo-CPU와 자체 simulator를 동일 조건에서 비교하는 평가 파이프라인을 구축했습니다. RL 성능을 간접 지표로 쓰는 대신 simulation과 real-world 간의 직접적인 일치도를 측정하는 Sim2Real metric을 제안해 채택되었습니다.

MODEL QUANTIZATION · 2024

Report ↗

PTMQ+

Multi-bit Weight & Activation Post-Training Quantization

소스코드가 공개되지 않은 PTMQ 논문을 PyTorch로 직접 구현해 block reconstruction, Multi-bit Feature Mixer, Group-Distill Loss를 재현하고 오픈소스로 공개했습니다.

activation 중심의 multi-bit PTQ를 weight까지 확장한 PTMQ-W를 제안하고, 최종적으로 가중치와 활성값 모두에서 가변 정밀도를 지원하는 PTMQ+를 개발했습니다. ResNet, MobileNet, RegNet, ViT를 ImageNet에서 검증했습니다.

EDGE ROBOT VISION · 2026

Delivery Robot Invoice VLM

Offline parcel-address recognition on Jetson Orin NX

배달로봇이 카메라로 택배 송장을 인식하고 동·호수·수취인 정보를 추출하는 end-to-end 파이프라인을 개발했습니다. YOLO OBB로 송장을 검출·크롭하고 InternVL3-2B를 LoRA 튜닝해 구조화된 필드를 생성했습니다.

ROS2와 Depth Camera를 이용한 거리 기반 autofocus를 구현하고, JetPack 6.1/7.2에 배포했습니다. 공식 이미지가 없던 7.2에서는 TensorRT-LLM을 직접 소스 빌드하고 SmoothQuant W8A8을 적용해 오프라인 0.5초 이내·정확도 98% 이상의 제약을 만족했습니다.
